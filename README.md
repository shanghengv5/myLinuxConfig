# 简介

因为有很多服务器要登录查看日志或者调试,所以需要有一个适合的自己的使用环境配置.
注意不要破坏原服务器的配置.

# myLinuxConfig安装

```bash
git clone https://github.com/shanghengv5/myLinuxConfig
bash myLinuxConfig/lsh_install.sh && . myLinuxConfig/scripts/lsh_env.sh
```
